# docs/ — source papers

PDFs are gitignored (large binaries). Drop the paper PDFs you want to process here, then
point the scripts at them.

Papers used during development:

- `2409.16118v3.pdf` — TabEBM (arXiv:2409.16118)
- `VARC.pdf` — Vision ARC (VARC)

Fetch a paper directly from arXiv, e.g.:

```bash
wget -O docs/2409.16118v3.pdf https://arxiv.org/pdf/2409.16118v3
```
